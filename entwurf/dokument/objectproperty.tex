\subsection{\label{objectproperty.GAnsProperty}\index{GAnsProperty}Class GAnsProperty}{
\vskip .1in 
The GAnsProperty is a single property of a vertex or an edge in GAns. The name and value of the property are being stored in JavaFX-Properties. A graph consists out of vertices and edges which contain GAnsProperties, which are linked to the GUI-elements.\vskip .1in
